# Customer App (Phase 6 — not started)

iOS/Android app for customers: browse the restaurant's menu and place a
preorder for pickup (pay at the store — no online payment until Phase 7,
and then only via processor-hosted checkout).

Talks to the restaurant's own Supabase project through the
`OnlineOrderChannel` contract in `packages/domain`. Restaurant discovery via
QR code / restaurant code.

Scaffolded in Phase 6 — see docs/ROADMAP.md.
