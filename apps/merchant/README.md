# Merchant App

The POS that runs on the restaurant's tablet: order taking, receipt printing,
payment terminal, reports. iOS first; Android/Windows capable.

Run it: `flutter run` from this directory (workspace resolves from the repo root).

Structure and layering rules: see [docs/ARCHITECTURE.md](../../docs/ARCHITECTURE.md)
and [lib/features/README.md](lib/features/README.md).
